What is the Pumping Lemma?

The Pumping Lemma (Statement)

If L is a regular language, then there exists a pumping length p ≥ 1 such that every string s ∈ L with |s| ≥ p can be written as s = xyz satisfying all three conditions:

|y| ≥ 1 |xy| ≤ p ∀i ≥ 0 : xyⁱz ∈ L

Proving Non-Regularity

To show L is NOT regular, we use a proof by contradiction:

  1. Assume L is regular with pumping length p.
  2. Choose a string s ∈ L with |s| ≥ p strategically.
  3. For every valid split s = xyz (|y|≥1, |xy|≤p)...
  4. Find some i ≥ 0 such that xyⁱz ∉ L.
  5. This contradicts the lemma → L is NOT regular.
Regular Languages: For languages like a*b* and (ab)+, the pumping lemma always holds — pumping never causes a contradiction because these languages genuinely are regular (accepted by finite automata). This tool demonstrates both cases side by side.

How the Pumping Lemma Works
01
Every regular language has a pumping length p. Any string in the language with length ≥ p can be split into x·y·z.
02
y is non-empty and |xy| ≤ p. y is always chosen from the first p characters — typically the earliest repeating section.
03
For a regular language, xyⁱz must stay in the language for every i ≥ 0. No exceptions allowed.
04
If pumping breaks membership for any i, we have a contradiction, and the language is NOT regular.
